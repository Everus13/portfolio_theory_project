# MOEX Portfolio Optimization (CatBoost + MPT + Sortino)

Проект по динамической оптимизации инвестиционного портфеля на Московской бирже и крипторынке с использованием машинного обучения (**CatBoost**), современной портфельной теории (**MPT / Модель Марковица**) и минимизации downside-рисков (**Sortino Ratio**).

Система прогнозирует ожидаемую доходность активов на неделю вперед, учитывает индекс волатильности Мосбиржи (**RVI**), ключевую ставку ЦБ РФ и находит оптимальные веса портфеля для максимизации коэффициента Сортино с учетом жестких инвестиционных границ.

---

## 💼 Инструменты и Структура Портфеля

В портфеле используются 4 класса активов с жесткими защитными лимитами (целевая аллокация: **40/35/15/10**):
* 📈 **TPAY** (Облигации с плавающей ставкой): доля от 25% до 45% (цель — 40%)
* 🟡 **TGLD** (Фонд Золота): доля от 20% до 35% (цель — 35%)
* 🪙 **BTC** (Биткоин в рублях): доля от 5% до 15% (цель — 15%, единственный рисковый актив-драйвер)
* 💵 **TMON** (Фонд денежного рынка): доля от 5% до 15% (цель — 10%)

---

## 💡 Идея проекта

Вместо традиционного использования исторической доходности активов (которая плохо прогнозирует будущее), мы применяем **прогнозы CatBoost Regressor**, обучающегося на:
1. Лагах доходностей отдельных активов.
2. Исторической волатильности инструментов.
3. Индексе страха российского рынка (**RVI**).
4. Динамике индекса акций Мосбиржи (**IMOEX**).

Ребалансировка запускается раз в месяц (20 торговых дней) или внепланово при дрейфе веса любого актива более чем на **5%**. Оптимизатор SLSQP находит веса, которые максимизируют коэффициент Сортино с учетом текущей безрисковой ставки ЦБ РФ (автоматически скачивается через SOAP API с cbr.ru).

## 🚀 Инструкция по запуску

### 1. Установка окружения (Python)
```bash
python -m venv .venv
# Активация на Windows:
.\.venv\Scripts\activate
# Установка библиотек:
pip install -r requirements.txt
```

### 2. Первоначальный сбор данных
```bash
python update_data.py
```

### 3. Настройка своего портфеля
Укажите количество лотов/паев в файле `portfolio_state.json`:
```json
{
    "TPAY": 392.0031,
    "TGLD": 2943.6501,
    "BTC": 0.003089,
    "TMON": 62.7274
}
```

### 4. Запуск веб-панели управления (FastAPI + React)

Для работы интерактивного Sleek Dark дашборда необходимо запустить два компонента:

**Шаг А: Запуск Backend (FastAPI)**
```bash
# Убедитесь, что виртуальное окружение активировано
python src/api/main_api.py
```
API-сервер запустится на порту **8000** (`http://127.0.0.1:8000`).

**Шаг Б: Запуск Frontend (React + Vite)**
```bash
cd frontend
# При первом запуске установите Node-модули:
npm install
# Запустите сервер разработки:
npm run dev
```
Интерфейс откроется на порту **5173** (`http://127.0.0.1:5173`).

---

### 5. Ежедневный запуск проверки (CLI)
Для автоматического контроля отклонений долей без веб-интерфейса запускайте:
```bash
python daily_check.py
```
Скрипт выведет точные торговые ордера на покупку/продажу активов в консоль.
