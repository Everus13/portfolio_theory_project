# MOEX Portfolio Optimization (CatBoost + MPT)

Проект по динамической оптимизации инвестиционного портфеля на Московской бирже с использованием машинного обучения (**CatBoost**) и современной портфельной теории (**MPT / Модель Марковица**).

Система прогнозирует ожидаемую доходность активов на неделю вперед, учитывая индекс волатильности Мосбиржи (**RVI**), и находит оптимальные веса портфеля для максимизации коэффициента Шарпа с учетом заданных ограничений.

---

## Идея проекта

Традиционная модель Марковица использует историческую среднюю доходность активов в качестве «ожидаемой». Однако историческая доходность плохо предсказывает будущее. 

В этом проекте мы заменяем историческую доходность на **прогнозы CatBoost Regressor**, обученного на:
1. Лагах доходностей отдельных активов.
2. Исторической волатильности активов.
3. Индексе волатильности Мосбиржи (**RVI**), отражающем уровень страха на рынке.
4. Доходности широкого рынка акций через индекс (**IMOEX**).

Каждую неделю портфель ребалансируется: оптимизатор SLSQP находит веса, которые максимизируют коэффициент Шарпа (предсказанная доходность / историческая недельная ковариация).

### Ограничения портфеля:
* Только длинные позиции (Long-Only, веса $\ge 0$).
* Сумма весов равна $1.0$ (100% капитала).
* Минимальный вес актива при инвестировании — **5%** (для избежания микро-долей).

## Запуск проекта

### 1. Установка окружения и зависимостей
Убедитесь, что у вас установлен Python 3.10+. Создайте виртуальное окружение и установите библиотеки:
```bash
python -m venv .venv
# Активация на Windows:
.\.venv\Scripts\activate
# Установка библиотек:
pip install -r requirements.txt
```

### 2. Загрузка данных
Запустите скрипт для скачивания истории котировок (акции: SBER, GAZP, LKOH; фонды: TPAY, TGLD; индексы: RVI, IMOEX) с MOEX API и записи их в SQLite:
```bash
python update_data.py
```

### 3. Обучение моделей CatBoost
Запустите процесс генерации фичей и обучения моделей для каждого актива:
```bash
python -m src.catboost_train.train
```

### 4. Запуск бэктестинга
Проведите симуляцию инвестирования с еженедельной ребалансировкой и постройте график доходности:
```bash
python backtest.py
```
